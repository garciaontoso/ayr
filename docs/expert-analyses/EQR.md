## ¿Qué hace EQR?

Equity Residential (NYSE: EQR) es **uno de los mayores apartment REITs USA** con sede en Chicago. Fundada 1969 por Sam Zell. Hoy con presencia en **9 markets premium coastal + Denver + Atlanta**, **~80,000 units**.

### Geografía

**Markets**:
- **New York / Tri-State** (NYC + suburbs)
- **Boston / New England**
- **Washington DC / Mid-Atlantic**
- **San Francisco Bay Area**
- **Seattle / Pacific Northwest**
- **San Diego**
- **Los Angeles + Orange County**
- **Denver**
- **Atlanta**

**Coastal premium concentration** similar a AVB pero subtle differences:
- EQR ligera mayor exposure NYC vs AVB
- EQR menos California Norte vs AVB
- EQR similar premium tenant base

### Portfolio composition

- **High-rise + Mid-rise** (~60%): urban premium
- **Garden-style** (~40%): suburban
- **Average rent**: $3,000+/month (premium)
- **Average occupancy**: 96%+ sostenido
- **80,000 units total**

### Sam Zell legacy

Founder Sam Zell (1941-2023) was **legendary REIT investor** — pioneered modern REIT structure 1990s, founded multiple successful REITs (EQR, ESL, EOP). Discipline + cycle awareness legendary.

Post-Zell death 2023, **Mark Parrell** (CEO since 2019) ejecutando bien continuing legacy.

### Estructura accionarial

- **Free float**: ~99%
- **Top institucionales**: Vanguard, BlackRock, State Street
- **Sam Zell estate**: small residual

CEO **Mark J. Parrell** desde 2019 — interno, ex-CFO. Continuidad estratégica desde David Neithercut era.

### El driver: coastal vs Sunbelt rotation

Same dynamic as AVB:
- 2020-2024: Sunbelt boom, coastal weak
- 2025+: coastal recovery, Sunbelt indigestion
- EQR benefits from rotation back to coastal

**Differential vs AVB**: EQR ligera exposure higher NYC = more sensitive return-to-office trend.
